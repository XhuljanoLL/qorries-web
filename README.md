# Qorri's Web

A personal portfolio site built with React, Three.js, Framer Motion, and TailwindCSS.

## Built With

- React 18
- Vite
- Three.js / React Three Fiber
- Framer Motion
- TailwindCSS
- EmailJS

## Getting Started

### Prerequisites

Node `18.0.0` or higher and npm `9.0.0` or higher.

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens the project on `http://localhost:5173/`

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
